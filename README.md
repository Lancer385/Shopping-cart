# Project: Shopping Cart

A simple shopping cart built with vite+react, you can browse products and add specific amount of them in the cart.

## Features

- Browse products on the Shop page
- Add items to cart with quantity control
- Cart updates in real time across the app
- cart icon badge showing total item count
- api products are fetched via [dummyJson](https://dummyjson.com/)

## Live Demo

[shoppingcartlancer.netlify.app](https://shoppingcartlancer.netlify.app)

**Note:** This project uses React 18 as it uses PropTypes for runtime type checking.
